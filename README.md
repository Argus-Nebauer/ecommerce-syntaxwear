# E-commerce SyntaxWear

## Descrição

Este é um site de e-commerce estático para a loja SyntaxWear, especializada em tênis e sneakers. O site apresenta uma interface moderna e responsiva, com seções para categorias de produtos, destaques e informações de contato.

## Funcionalidades

- **Página Inicial**: Apresenta um banner hero, categorias de tênis (Casual, Esporte, Moderno, Futurista), grid de produtos em destaque e rodapé com newsletter e links.
- **Navegação**: Menu responsivo com categorias principais e acesso rápido a conta, ajuda e carrinho.
- **Design Responsivo**: Otimizado para dispositivos móveis e desktop.
- **Newsletter**: Formulário para inscrição em e-mails.
- **Redes Sociais**: Links para Instagram, WhatsApp, TikTok e Facebook.

## Tecnologias Utilizadas

- **HTML5**: Estrutura do site
- **CSS3**: Estilização e layout responsivo
- **SVG**: Ícones vetoriais

## Como Executar

1. Clone ou baixe o repositório.
2. Abra o arquivo `index.html` em um navegador web moderno.
3. O site será carregado e você poderá navegar pelas seções.

Não há necessidade de servidor, pois é um site estático.
